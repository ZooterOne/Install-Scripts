<h1 align="center">Install Scripts</h1>

# :thinking: About

Various scripts and configuration files including:

  * Fedora installation & setup.
  * Kali installation & setup.
  * Alacritty configuration.
  * Eza configuration.
  * Fastfetch configuration.
  * Grub configuration.
  * Starship configuration.
  * Visual Studio Code configuration.

# :computer: Use

Run the following:

``` bash
./setup-<name>.sh
```

Then follow the prompt to setup environment or to install and configure additional tools.
